# Code of Conduct

## Our Standards

This project welcomes participation from everyone. We ask that all contributors:

- Be respectful and considerate in all interactions
- Focus feedback on ideas and code, not on individuals
- Accept constructive criticism gracefully
- Support a welcoming environment for contributors of all backgrounds and experience levels

## Enforcement

Instances of unacceptable behavior may be reported by contacting the maintainer at
[chicken@chickenandstats.com](mailto:chicken@chickenandstats.com). All reports will be
reviewed and handled with discretion.

## Scope

This Code of Conduct applies to all project spaces, including GitHub issues, pull requests,
discussions, and any other communication channels associated with `chickenstats`.
